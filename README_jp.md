# ScrapboxPortalLink
Scrapboxで特定プロジェクトを利用中に、他のプロジェクトのページとリンクをさせます。

## Installation

	$ npm install
	$ npm run dev:chrome
 
### on GoogleChrome
  - アクセス -> (chrome://extensions/)
  - developerモードをONにします。
  - 「パッケージ化されていないプラグインを読み込む」をクリックし、以下のディレクトリからエクステンションを読み込みます。 -> (/dist/chrome/)
  - 詳細、オプションで、リンクさせたいプロジェクトのURLを設定します。
